# POSflyt Mobile App Scope (React Native Preparation)

## Primary Modules
- POS: product search, cart, checkout, receipts, offline queue.
- Dashboard: business KPIs, alerts, forecast snapshots.
- Inventory: create/update products, stock view, low-stock handling.

## API Expectations
- JWT auth with role and plan context.
- Consistent JSON responses and standard error payloads.
- Offline-safe transaction sync endpoint.

## Out of Scope (Initial Native Release)
- Full billing management UI
- Deep admin-only platform metrics
- Advanced back-office configuration workflows
